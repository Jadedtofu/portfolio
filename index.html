Projects:

Flavor Logs

An application for food lovers to keep a record of dishes they've tried at various eateries, old or new. Users are able to add, edit, and delete Flavor Logs as well as Eateries. An image of the dish tried can be added to the log to further enhance its description. Created using React, CSS, Node.js, Express, CSS using a PostgreSQL database.

Chore Share

A simple application for roommates to share chores in one place. Roommies and Chores can be added or deleted while Chores can be marked complete. When a Roomie is removed, all the Chores relating to that Roomie will be removed. Created using React, CSS, Node.js, Express, CSS with a PostgreSQL database.

Programming Focus

A search application meant for new programmers. It utilizes the Stack Overflow and YouTube APIs to assist in a focused search to further assist developing programmers through the learning process. Search results may vary depending on the keywords used. Created using HTML, CSS, JavaScript, and jQuery
